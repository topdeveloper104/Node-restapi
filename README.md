Simple Express api.

- Set up

  - git clone.
  - npm install
  - node server

- Use

  - Get all models    ---     GET : api/models
  - Get a model       ---     GET : api/models/_id
  - Create new model  ---     POST : api/models
  - Update a model    ---     PUT : api/models/_id
  - Delete a model    ---     DELETE : api/models/_id
  - Search            ---     GET :api/search?key={key}
